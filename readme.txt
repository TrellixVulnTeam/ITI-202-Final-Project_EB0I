How to run program:

1. Ensure that the required packages from requirements.txt are installed
2. run main.py
3. Open a web browser and go to http://127.0.0.1:5000
4. Enter the @ username you wish to analyze tweets from (Please not the program will create an error if the account is private or does not exist) (You may also click the suggested accounts!)
